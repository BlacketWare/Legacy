  <p>1. Copy the script to your clipboard.</p>
  <p>2. Open the console in Betastar (CTRL + SHIFT + J or COMMAND + SHIFT + J).</p>
  <p>3. Paste the script in (CTRL + V).</p>
  <p>4. Press enter.</p>
  <p>5. Follow any instuctions in popups.</p>
