# How to use
1. Copy the script to your clipboard.<br>
2. Open the console in Betastar (CTRL + SHIFT + J or COMMAND + SHIFT + J).<br>
3. Paste the script in (CTRL + V).<br>
4. Press enter.<br>
5. Follow any isntructions in popups.
